Tell Me More


Who Are We? We are computer scientists who study the foundations of information systems and their applications. We are working on small- and large-scale projects to help individuals and industries to use smart sensor technologies easily. The overarching goal of our research is to make theoretical and practical contributions to challenging and current problems in the domain of collaborative and competitive interactions in distributed and intelligent systems, with a specific emphasis on the Internet of Things (IoT) and Sensor-based Systems.


“Dr. Ahmed leads the IoT Research Lab. Dr. Ahmed is an assistant professor at the Department of Computer Science in Prairie View A&M University, which is a member of the Texas A&M University System.”

IoT Infrastructure

Building an IoT Ecosystem that can be widely deployed in various domains

Building an open IoT platform that can be widely deployed and used by building a new, robust and integrated stack with innovations at the hardware, software and end-user levels.

IoT Applications

Developing Smart IoT Applications using Cutting-Edge IoT Technologies

Building a complete IoT ecosystem encapsulates a high-performance big data platform that can be used by users for initiating and managing innovative IoT applications using a simple GUI.

Sensor Data Analytics

Real-time Data Analytics for Live Sensor Data Streams

Leveraging deep learning to support real-time decision making in IoT applications including, smart agriculture, health monitoring, traffic management, autonomous driving and more.

Autonomous UAVs

Building Robotic Scout UAVs for Pipeline Corrosion Detection

Building a drone-like robot (Unmanned Arial Vehicle) which can scout for damage inside pipelines maintained by the Bechtel Corporation – The largest construction company in the United States.

IoT Services

Developing Algorithms, Formal Models and Software Packages IoT Services

Building several algorithms, formal models and software packages to support the programmability and performance of sensor-based IoT services.

Smart IoT Irrigation

Developing Smart Irrigation Systems using IoT Technologies for Smart Farming

Developing smart irrigation systems using IoT technology which can be used for automating the irrigation process in agricultural fields. This is a collaborative project with the College of Agriculture and Human Sciences (CARC) in PVAMU.

Travel Industry

Developing Resource Control Mechanisms for the Cloud-Based Travel Industry

Implementing scalable solutions for supporting resource control in multi-tenant cloud-based systems for the travel industry as an extension for Scala/Akka programming language.

IoT Cybersecurity

Developing Robust and Secure Algorithms, Frameworks, Solutions for IoT Systems

Developing and evaluating several security solutions for intelligent sensing and distributed systems such as IoT using deep learning algorithms to detect security vulnerabilities that target these systems.

Secure MANETs

Developing Novel Secure Routing Protocol for Mobile Ad-hoc Networks (MANETs)

Developing novel secure routing protocols for mobile ad-hoc networks (MANET), which significantly reduced the number of attacks on the network.
